# GISFaces

Enterprise GIS Component For JavaServer Faces

## Maven
- Build Jar: mvn clean install
- Build Source Jar: mvn source:jar
- Build JavaDoc Jar: mvn javadoc:javadoc
- Format Code: mvn spotless:apply
